# CPP Playground

Recording some cpp practice and experiments.

## Dependencies

- **C++ compiler** — Clang-cl (MSVC-compatible command line), C++23
- **Build system** — CMake 3.28+ with Ninja
- **Google Test** — fetched automatically via CMake FetchContent (test targets only)
- **Python** — 3.12+, matplotlib, seaborn, pandas (plot script only)
